# Level 2 — SACCO Member Financial Record System

## Description
This project calculates total savings for a SACCO member based on monthly contributions.
It is implemented in Python and C++.

## Concepts Demonstrated
- Static vs Dynamic Typing
- Type Coercion
- Primitive vs Composite Types

## Usage
1. Enter member name and ID.
2. Enter monthly contributions for 6 months.
3. Program outputs total savings.

## Code Files
- `level2_sacco.py` (Python)
- `level2_sacco.cpp` (C++)
